# 2. Oracle 12c(12.2) 소프트웨어 설치

### 1. OS 사전 설정(Pre-install)

#### 호스트명 변경

```bash
[root@localhost ~]# hostnamectl set-hostname oracle12c
[root@localhost ~]# hostname
oracle12c
[root@localhost ~]# cat /etc/hostname
oracle12c

[root@localhost ~]# vi /etc/hosts
[root@localhost ~]# cat /etc/hosts
127.0.0.1   localhost localhost.localdomain localhost4 localhost4.localdomain4
::1         localhost localhost.localdomain localhost6 localhost6.localdomain6
192.168.13.81 oracle12c
```

#### 12c preinstall 패키지 설치

```bash
yum install -y oracle-database-server-12cR2-preinstall
```

#### 필수 그룹 생성 여부 확인

```bash
[root@localhost ~]# cat /etc/group
...
oinstall:x:54321:
dba:x:54322:
oper:x:54323:
backupdba:x:54324:
dgdba:x:54325:
kmdba:x:54326:
racdba:x:54330:
```

#### oracle 유저 그룹 설정(oinstall + 추가 그룹)

```bash
[root@localhost ~]# id oracle

[root@localhost ~]# usermod -g oinstall oracle
[root@localhost ~]# usermod -a -G dba,oper,backupdba,dgdba,kmdba,racdba oracle

[root@localhost ~]# id oracle
```

#### 커널 파라미터 적용 확인

```bash
[root@localhost ~]# sysctl -p
```

#### 리소스 제한(limits) 설정 확인

```bash
[root@localhost limits.d]# grep -v ^# /etc/security/limits.d/oracle-database-server-12cR2-preinstall.conf
```

---

### 2. 디렉토리 생성 및 권한 설정

```bash
[root@localhost ~]# mkdir -p /u01/app/oracle
[root@localhost ~]# mkdir -p /u01/app/oraInventory
[root@localhost ~]# mkdir -p /u01/app/oracle/product/12.2.0/dbhome_1

[root@localhost ~]# chown -R oracle:oinstall /u01
[root@localhost ~]# chmod -R 775 /u01
```

---

### 3. oracle 계정 환경변수(.bash_profile) 설정

```bash
[root@localhost ~]# su - oracle
[oracle@oracle12c ~]$ vi .bash_profile
```

예시:

```bash
export ORACLE_BASE=/u01/app/oracle
export ORACLE_HOME=/u01/app/oracle/product/12.2.0/dbhome_1
export ORA_INVENTORY=/u01/app/oraInventory
export ORACLE_SID=ORA12
export TNS_ADMIN=/u01/app/oracle/product/12.2.0/dbhome_1/network/admin
export PATH=$PATH:$HOME/bin:$ORACLE_HOME/bin
export LD_LIBRARY_PATH=$ORACLE_HOME/lib:/lib:/usr/lib
export CLASSPATH=$ORACLE_HOME/JRE:$ORACLE_HOME/jlib:$ORACLE_HOME/rdbms/jlib
export NLS_LANG=KOREAN_KOREA.AL32UTF8

alias sys='sqlplus / as sysdba'
alias scott='sqlplus scott/tiger'
alias net='cd /u01/app/oracle/product/12.2.0/dbhome_1/network/admin/'
```

적용 확인:

```bash
[oracle@oracle12c ~]$ source .bash_profile
[oracle@oracle12c ~]$ env | grep ORA
```

#### root로 접속하여 스크립트 실행

- 터미널을 열고 root로 접속
- 아래 스크립트를 실행하고, 기본값이면 엔터로 진행

```bash
[root@oracle12c ~]# /u01/app/oraInventory/orainstRoot.sh

[root@oracle12c ~]# /u01/app/oracle/product/12.2.0/dbhome_1/root.sh
# Enter the full pathname of the local bin directory: [/usr/local/bin]:
# Do you want to setup Oracle Trace File Analyzer (TFA) now ? yes|[no] :
```
